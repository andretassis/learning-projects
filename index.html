Como é uma estrutura básica de um HTML?

Início

O HTML é uma linguagem de marcação que será interpretada por navegadores (browsers).

A estrutura básica se dá por meio de tags, em sua maioria com aberturas e fechamentos*, que mostrarão o conteúdo de diferentes formas a depender do seu uso.

Existem duas principais tags que dão forma e corpo à estrutura do HTML: < html > e < body >. Todo o conteúdo que será mostrado ao usuário está no corpo do documento, ou seja, na tag < body > que, por sua vez, está contido na tag estrutual do documento, a < html >.

A estrutura básica fica então assim:

< html >
< body >
Conteúdo do site
< /body >
< /html >

*: as tags de fechamento são iniciadas por " / " e estão escritas ao final. Algumas tags não possuem fechamento, como por exemplo "< img >".

Entendendo as tags gerais

Início

Chamaremos de tags gerais as tags utilizadas para delimitar algumas partes do texto. São elas:

  • < title >: muda o título que aparece na aba do usuário
  • < h1 > a < h6 >: coloca um título na página. O tamanho e destaque do título se dá de acordo com o número, sendo h1 o maior e o h6 o menor.
  • < p >: inicia um parágrafo
  • < ul >: utilizada para criar uma lista sem ordem específica. Neste caso, o marcado são figuras, como bolinhas.
  • < ol >: utilizada para criar uma lista com ordem específica. Neste caso, o marcador será numérico
  • < li >: ao criarmos uma lista, seja ela com ou sem ordem, essa tag será necessária para listar os elementos.
  • < a >: colocar um hyperlink na página.

Entendendo as tags específicas

Início

Chamaremos de tags específicas as tags utilizadas para caracterizar o texto. São elas:

  • < strong >: coloca as palavras ou texto em negrito.
  • < mark >: destaca alguma parte do texto.
  • < i >: colocar as palavras ou texto em itálico.
  • < u >: sublinhar as palavras ou texto.
  • < sub >: escreve algo abaixo da linha, como o numeral 2 em: A2
  • < sup >: escreve algo acima da linha, como o asterisco neste exemplo: exemplo*
  • < blockquote >: utilizado para fazer alguma citação no texto

Entendendo as listas ordenadas e não ordenadas

Início

Existem dois tipos de listas: as ordenadas e as Desordenadas.
As listas ordenadas são aquelas em que o marcador é numérico e, como o próprio nome diz, representa uma ordem:

  1. Primeiro lugar
  2. Segundo lugar
  3. Terceiro lugar

As listas não ordenadas possuem os marcadores em figuras, como bolinhas ou quadradinhos e não representam necessariamente uma ordem:

  • Laranjas
  • Bananas
  • Maçãs
